# GameDevProject

Open project in Unity version 6.2.

Project starter used: 3D (Built-In Render Pipeline)

## How to Run

Open `CanyonRun` -> `CanyonRun.exe`

If this does not work, open the `CanyonRun` file as a project in Unity 3D.

## Assets used
